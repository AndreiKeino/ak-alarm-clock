Quickstart:

  1. Download appropriate appllication build and copy it to the location of your choice. You can also rename this file.
  2. To start the application, doubleclick the downloaded file.
  3. To hide the main window, click the button "x" in the topright corner of the window.
  4. To show the main window either doubleclick the taskbar icon or choose the taskbar menu item "Show".
  5. To add an alarm, use the main window button with green cross in the topleft part of the main window.
  6. To remove an alarm, use the main window red circle button with white minus sign in the topleft part of the main window.
  7. To edit an alarm, doubleclick it in the alarm list.
  8. To exit the application, choose the taskbar menu item "Close".
  9. Enjoy.
